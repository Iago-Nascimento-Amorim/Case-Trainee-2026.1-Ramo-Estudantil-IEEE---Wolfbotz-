# Case-Trainee-2026.1-Ramo-Estudantil-IEEE---Wolfbotz-
Robô automotivo com modo controle remoto (controlado pelo App 'Arduino Bluetooth Controller), e modo de estacionamento autônomo. ESP32 com arquitetura FreeRTOS dual-task, sensores HC-SR04, ponte H TB6612FNG e motores N20. Detecção geométrica de vaga e manobra automatizada.
